What's the difference between Grid and Flexbox?

These boxes are aligned with display: grid!

.bigbox-grid {
display: grid;
grid-template-columns: 52px 52px 52px;
}

These boxes are aligned with display: flex!

.bigbox-flex {
width: 158px;
display: flex;
align-content: center;
justify-content: center;
flex-wrap: wrap;
}
